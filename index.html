Welcome to CS125 Course Companion!

What is CS125 Course Campanion?

CS125 Course Companion is a site that aims to share and provide academic resources and extra materials for students who is currently taking or is planning to take CS125. This site is run by a group of students who are currently taking the course and is willing to share their notes and resources for future students.

In this site, we have extra resources organized in the following sections:

Java Basics
In this part you will begin your journey, learn the fundamentals of the Java language. It's really important that you learn this section well!
Object Oriented Programming (OOP)
Java is an object oriented programming language. This section probably has the most conceptual challenges, but it's also fun!
Algorithms
Algorithms lie at the core of problem solving. Not only for computer science, but other subjects and day-to-day problems. Furthur sutdy on this topic will be introduced in CS374.
Data Structures
Data structures is a study that researches on how to most effectively assemble data. Java Collection Framework provides a good support for many data structures. Furthur study on this topic will be introduced in CS225.
Additional Resources
This section contains many useful resources that can be referenced to help you better study for CS125!
Machine Project (MP)
MPs are challenging but fun! You will learn a lot about Java and Android through the process, and we will provide you with some resources on these topics.
Bonus
If you are really curious and is willing to learn more about Java beyond what is required for this course, check out resources here!
