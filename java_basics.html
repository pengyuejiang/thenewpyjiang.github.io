1. Types and Variables

Variable declaration

In Java, as in any other programming languages, data are stored in variables. These things act like containers, holding your data. In Java, whenever we want to create such a container, we have to declare a variable with its type followed by variable name, for instance:

// Declare a varible called age as an integer
int age;
// Declare a variable called length as a float
float length;
// Declare a variable called grade as a character
char grade;

Variable initialization

However, at this stage we cannot use them yet, because they do not have an initial value yet. The process where we initialize their values is called initialization. See the code section below:

// Declare int variable age
int age;
// Initialize it to be 5
age = 5;

It is much more common to see declaration and initialization done together in a line:

// Declare int variable age and initialize it to be 5
int age = 5;
